# 📚 EduGenie AI

### AI-Powered Learning Assistant Prototype

EduGenie AI is a prototype educational web application designed to help students and teachers generate AI-powered summaries, quizzes, and interactive learning support from study materials.

This project combines frontend web development with AI-inspired educational workflows to create a clean and user-friendly learning platform.

---

# 🚀 Features

✅ AI Summary Generator  
✅ Quiz Generator  
✅ AI Tutor Chatbot  
✅ Interactive Web Interface  
✅ Responsive UI Design  
✅ Educational Workflow Prototype
